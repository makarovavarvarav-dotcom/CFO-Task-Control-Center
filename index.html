Быстрый старт

  1. Откройте `index.html` двойным кликом в браузере.
  2. Перейдите во вкладку «Журнал/настройки».
  3. Нажмите «Создать демо-данные», если база задач пустая.
  4. Для рабочей эксплуатации создавайте задачи вручную через «Список задач».
Приложение полностью локальное: база хранится в LocalStorage браузера под ключом `cfoTaskControlCenterDB`.

Создание и редактирование задач

Во вкладке «Список задач» нажмите «Создать задачу». Заполните обязательные поля: название, категория, статус, приоритет и дедлайн.

CEO-фокус

Если задача отмечена как CEO-фокус, перенос дедлайна требует обязательную причину. Без причины сохранение будет заблокировано.

Повторение

Отметьте «Повторение» и выберите правило: ежедневно, еженедельно, ежемесячно, ежеквартально или ежегодно. При загрузке страницы система создаст один новый экземпляр без дублей.

Дашборд дня

  • Верхние карточки показывают активные задачи, дедлайны на сегодня, просрочки, риски, P1 и CEO-фокус.
  • «Что делать сегодня» выводит до 5 задач с максимальным score.
  • «Горит» показывает просроченные задачи.
  • «Фокус CEO» собирает активные задачи с флагом CEO.

Канбан

Перетащите карточку между колонками. После сброса статус задачи меняется автоматически, дата обновления пересчитывается, а действие записывается в журнал.

Клик по карточке открывает форму редактирования.

Матрица и риск-карта

Матрица

Матрица раскладывает активные задачи по четырем квадрантам на основе приоритета и срочности.

Риск-карта

Риск-карта группирует активные задачи по типу риска и показывает счетчики: всего, P1, просрочено, critical.

Экспорт, импорт и восстановление

  • «Экспорт базы» скачивает полный JSON-бекап.
  • «Импорт слиянием» добавляет новые задачи и обновляет существующие только если импортная версия новее.
  • «Полная перезапись базы» требует точное контрольное слово `ПЕРЕЗАПИСАТЬ`.
Перед полной перезаписью рекомендуется сделать экспорт текущей базы.
